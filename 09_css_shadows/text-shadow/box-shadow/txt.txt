!Umbrele elementelor!

--> Mai exact, este vorba de umbrele care se adaugă pe cutiile imaginare (box-urile) în cadrul cărora se introduc elemente HTML.
 Tocmai de aceea, pentru crearea acestor umbre se foloseşte proprietatea box-shadow.

//box-shadow: inset offset-x offset-y blur-radius spread-radius color;

--> Semnificaţia diferitor valori care se pot găsi în cadrul proprietăţii box-shadow este ilustrată mai jos:

*offset-x --> distanţa umbrei de la element pe axa x --> da --> 	/
*offset-y --> distanţa umbrei de la element pe axa y --> da --> 	/
*blur-radius --> intensitatea estompării --> nu --> 0
*spread-radius --> dimensiunea umbrei --> nu--> 0
*color --> culoarea umbrei --> nu --> depinde de browser
*inset --> tip de umbră drop-shadow sau inset --> nu --> drop-shadow

*Poziţia umbrei*

-->  box-shadow impune definirea a cel puţin două valori pentru distanţa umbrei de la elemente, pe axele x şi y:

//box-shadow: 12px 12px;


*Estomparea umbrei*

--> Dacă pe lângă valorile utilizate deja specificăm şi a treia valoare, aceasta va reprezenta intensitatea estompării:

//box-shadow: 12px 12px 6px;


*Dimensiunea umbrei*

--> A patra valoare a proprietăţii box-shadow se referă la dimensiunea umbrei:

//box-shadow: 12px 12px 6px 12px;


*Culoarea umbrei*

--> Culoarea umbrei elementului se poate influenţa prin ultima valoare a proprietăţii box-shadow:

//box-shadow: 12px 12px 6px 6px #d6d6d6;


*Tipul de umbră*

--> Implicit, umbrele elementelor se poziţionează după elementul propriu-zis, iar acest tip de umbre se mai numeşte şi drop shadow.
 Pe lângă acest tip implicit de umbre, CSS permite şi crearea aşa-numitelor umbre inset. Acestea sunt umbre care se poziţionează în cadrul elementului propriu-zis. 

//box-shadow: inset 0 0 14px #2d111b;
//box-shadow: 0 0 14px #2d111b;
